# Diretrizes do Projeto

## Regras de Interface e UX
- **Preservação de Componentes Visuais:** Nunca remova componentes de identidade visual, como logotipos de companhias aéreas (`AirlineLogo`), a menos que seja explicitamente solicitado.
- **Fidelidade ao Layout:** Ao habilitar novas funcionalidades (como edição estilo planilha), as melhorias devem ser integradas ao design existente, nunca substituindo elementos gráficos por texto puro sem aviso prévio.
- **Sincronização Bilateral:** Alterações na Malha Geral e Malha Base devem ser refletidas mutuamente.

## Idioma
- Toda a comunicação com o usuário e labels de interface devem preferencialmente estar em **Português (PT-BR)**.
